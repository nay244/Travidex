Icon + mono value + label tile.

```jsx
<StatTile icon={<Footprints size={16}/>} value="Easy" label="Access" />
<StatTile icon={<Maximize size={16}/>} value="Large" label="Size" />
<StatTile icon={<Users size={16}/>} value="Busy" label="Busyness" />
```

Use as a 3-up strip on Sight Detail (Access · Size · Busyness — the geocaching D/T/Size analog) and as a Profile stats grid (sights found, cities claimed, countries, badges).
